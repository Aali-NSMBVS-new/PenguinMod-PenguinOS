# PenguinMod-PenguinOS

### thank you for contributing to this project!
### keep in your mind, that:
* it's not a real OS
* this is built with Scratch-based blocks
* some features may be on progress, experimental or bricked

### incase of that, we will give you tips:


#### HOW TO CONTRIBUTE TO THIS?
* fork it
* make a branch :
  ```
  PenguinMod-PenguinOS/fixes/name
  PenguinMod-PenguinOS/updates/name
  ```
* make changes
* test thoroughly
* commit with a clear message
* pull a request
