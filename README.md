# PenguinMod-PenguinOS
## PenguinOS is a PenguinMod OS made by aalinsmb!

#### WHAT DOES IT USE?
basically, **it** uses:
* custom extensions
* ChatGPT for idea

#### HOW TO USE IT?
##### for PMP
* first, download **PenguinOS.pmp** file
* open it in [PenguinMod - Editor](https://studio.penguinmod.com/editor.html)
* import the **PenguinOS.pmp** file
##### for HTML
* first, download the **PenguinOS.html** file
* view the **PenguinOS.html** file in your browser

that's how you can use **it**!
